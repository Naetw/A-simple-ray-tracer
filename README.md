# A-simple-ray-tracer

NSD Final Project.

## Build & Test

1. Execute CMake

```bash
mkdir build
cd build
cmake ..
```

2. Execute make

```bash
make
```

3. Run test

```bash
ctest
```

If there's any failed test, just execute the executable of that test directly to see the cause of failure.
Executables of unittests are prefixed with "unittest_".
